# プロダクト仕様

いろころは、保護者が赤ちゃんに語りかけながら色・かたちを 1〜3 分だけ楽しむ
短時間の補助ツール。絵本や人との関わりの代替ではない。

## 安全上の位置付け

- 初期時間は 1 分、音は常に OFF
- 一時停止と終了は、再生中いつでも操作できる
- 終了後に自動再生・自動リピートしない
- 点滅、急な白画面、赤色フラッシュを使わない
- 再生中の色・かたちは常に動かす（OS の `prefers-reduced-motion` でも抑えない）
- 広告、分析 SDK、Cookie、外部 CDN、個人情報収集、バックエンド通信はない
- 1 歳未満の座位でのスクリーン時間を勧めるものではない。保護者がそばで見守り、短時間で終了する

## セッション要件

| 項目 | 要件 |
|------|------|
| 初期値 | pack=`colors`、duration=`60`、sound=`false` |
| 時間選択肢 | 60 / 120 / 180 秒 |
| あそび | いまは `colors` のみ。`animals` は枠のみ（未実装） |
| シーン dwell | 6–8 秒。隣接シーンで色・形を連続させない |
| 背景クロスフェード | 約 900ms |
| 図形の動き | 上下左右・斜めからコロコロ転がり入場し、その後も転がり続ける。同一色の図形を複数出してよい |
| 音（任意） | 転がりのタイミングで「ころころ〜」系の合成音。切替チャイムではない |
| 一時停止 | カウントダウンとシーン進行の両方を止める |
| 終了 | 明示操作または期限到達。リセットするまで再開しない |

## 非スコープ（MVP）

アカウント、同期、コンテンツ CMS、外部音源、自動リピート、広告・課金。

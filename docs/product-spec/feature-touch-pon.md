# 機能仕様: タッチ反応「ぽん」

再生中のシーンを触ると、近い図形（またはどうぶつ）が短く膨らんで戻る。
「触ったら動いた」という因果を、セッションの進行を変えずに返す。

## 目的

- 保護者が赤ちゃんの手を添えたり、赤ちゃん自身が触れたときに、穏やかな反応を返す
- 触覚と視覚の対応を、語りかけのきっかけ（「ぽん」「動いたね」）として使えるようにする
- 点滅や強いフラッシュで驚かせず、既存モーションの延長として感じられる反応に留める
- 一時停止・終了・時間管理の安全性を壊さない

## なぜこの反応か

| 選択 | なぜ |
|------|------|
| playing のみ反応する | Setup / 一時停止 / 終了では「あそび中」ではない。誤反応で区切りを曖昧にしない |
| スケールの「ぽん」（色変化なし） | 因果は伝わるが、フラッシュや点滅に見えない |
| ピーク約 1.35 倍・短い一往復 | 触った感ははっきりしつつ、語りかけの邪魔にもならない |
| 全体 250ms に 1 回まで | 連打でチカチカ・連続バウンドに見えるのを防ぐ |
| セッション FSM に触れない | 触ったことでシーン送りや時間短縮が起きると、保護者の意図した区切りが崩れる |
| Setup に ON/OFF を置かない | 設定を増やさず、触れたときの安心できる反応として常に用意する |
| 初回は colors から | いちばん使われるあそびで因果を先に届ける。animals 実装後は同じ意図で両 pack に広げる |

## 保護者体験としての意図

「ぽん」はゲームのスコアや正解判定ではない。
触れた対象を一緒に見て、「動いたね」「まるが おおきくなったね」と言える余白を作る。

## 安全との関係

- 色の急変・白フラッシュ・赤フラッシュは使わない
- 短周期の明滅やサイズ点滅に見える連打反応を、レート制限で防ぐ
- コントロール（一時停止・終了）への操作は scene に伝播させない
- 効果音は初回スコープ外（不意の音刺激を増やさない）

## 受け入れの観点

- playing 中に scene をタップすると、近い図形が短く膨らんで戻る
- paused / finished / Setup では反応しない
- 連打しても点滅や連続バウンドに見えない
- 一時停止・終了・残り時間・シーン切替の挙動が変わらない

## 非スコープ（初回）

スワイプで投げる、ドラッグ追従、タップでシーン送り／色変更、効果音、
Setup での ON/OFF、パーティクルやフラッシュ、長押し・ダブルタップの別アクション。

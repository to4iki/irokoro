# 機能仕様: 図形の転がりモーション

再生中の図形を静止画ではなく動かし続ける。目的は演出の派手さではなく、
赤ちゃんの注意を穏やかに保ち、保護者の語りかけ時間を稼ぐことにある。

## 目的

- 画面が「止まって見終わった」感じにならないようにする
- 視線が図形を追いやすくし、保護者が指さしや声かけを続けやすくする
- 動きそのものを主目的にせず、語りかけの背景として控えめに保つ

## なぜ動かすか

| 選択 | なぜ |
|------|------|
| 常に動かす（reduced-motion でも抑えない） | 乳児向け補助では「止まった画面」より、ゆるい動きの方が注意を保ちやすい。ここではアクセシビリティの一般則よりプロダクト意図を優先する |
| 入場 → 大きな軌道 | 登場で注意を引き、その後もシーン全域を横断して「まだ見られる」状態を作る |
| 複数アクター（主＋副） | 主対象をはっきり見せつつ、画面がさびしくならない |
| 回転を移動に合わせる | 転がり感があり、形の輪郭が単調に固定されない |
| 点滅や急な拡大縮小をしない | 動きで注意を引いても、てんかんリスクや恐怖刺激は避ける |

## 保護者体験としての意図

モーションは「すごいアニメ」ではなく、「いまこれを見てるね」と一緒に言える余白を作る。
一時停止したら動きも止めるのは、中断時に世界が進行し続けない安心のため。

## 受け入れの観点

- playing 中、図形が入場したあとシーン内を大きく動く
- paused / タブ非表示では動きが止まる
- 点滅・フラッシュ・急転回に見えない

## 非スコープ

スプライトアニメ、物理エンジン、ユーザー操作で軌道を変えること（タッチ反応は別仕様）。

# プロダクト仕様

いろころは、保護者が赤ちゃんに語りかけながら色・かたちを 1〜3 分だけ楽しむ
短時間の補助ツール。絵本や人との関わりの代替ではない。

仕様ドキュメントは **なぜその体験にするか（目的）** を先に書く。
実装の置き場所や描画手順は [architecture.md](../architecture.md) 側に寄せる。

## プロダクトの目的

- 保護者と赤ちゃんが、同じ画面を短い時間だけ一緒に見るためのきっかけを渡す
- 画面が主役にならず、声かけ・指さし・見守りの補助にとどまる
- 長時間視聴や自動再生で引き込まず、終わったら人の側へ戻しやすくする

## 安全上の位置付け（なぜこう縛るか）

| 制約 | なぜ |
|------|------|
| 初期 1 分・音 OFF | 刺激と時間を、保護者の意思なしに増やさない |
| いつでも一時停止・終了 | 「止められない不安」を減らす |
| 自動リピートしない | 見終わったあとに再び引き込まない |
| 点滅・急な白画面・赤フラッシュ禁止 | 強い視覚刺激とてんかんリスクを避ける |
| 広告・解析・外部 CDN・個人情報収集なし | 赤ちゃん向け短時間ツールに、追跡や外部依存を持ち込まない |
| 1 歳未満の座位スクリーン時間を勧めない | 医療・養育の指針に逆らわない。そばでの見守りが前提 |

## セッション要件（要約）

| 項目 | 要件 |
|------|------|
| 初期値 | pack=`colors`、duration=`60`、sound=`false` |
| 時間選択肢 | 60 / 120 / 180 秒 |
| あそび | `colors` と `animals`。初期 pack は `colors` |
| シーン dwell | 6–8 秒。隣接シーンで色・形（どうぶつ）を連続させない |
| 色 | 原色多めの高コントラスト。点滅や赤フラッシュは使わない |
| 動き | 再生中は動かす。一時停止・タブ非表示では止める |
| 音 | 任意のループ BGM。初期 OFF。セッション単位で 1 曲 |
| 終了 | 明示操作または期限到達。リセットするまで再開しない |

## 非スコープ（MVP）

アカウント、同期、コンテンツ CMS、自動リピート、広告・課金。
音源の二次配布・販売や、音楽聴取を主目的とした配信は行わない（利用条件は [music-credits.md](../music-credits.md)）。

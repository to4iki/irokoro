# プロダクト仕様

いろころは、保護者が赤ちゃんに語りかけながら色・かたちを 1〜3 分だけ楽しむ
短時間の補助ツール。絵本や人との関わりの代替ではない。

## 安全上の位置付け

- 初期時間は 1 分、音は常に OFF
- 一時停止と終了は、再生中いつでも操作できる
- 終了後に自動再生・自動リピートしない
- 点滅、急な白画面、赤色フラッシュを使わない
- 再生中の色・かたちは常に動かす（OS の `prefers-reduced-motion` でも抑えない）
- 広告、分析 SDK、Cookie、外部 CDN、個人情報収集、バックエンド通信はない
- 1 歳未満の座位でのスクリーン時間を勧めるものではない。保護者がそばで見守り、短時間で終了する

## セッション要件

| 項目 | 要件 |
|------|------|
| 初期値 | pack=`colors`、duration=`60`、sound=`false` |
| 時間選択肢 | 60 / 120 / 180 秒 |
| あそび | `colors` と `animals`。初期 pack は `colors` |
| シーン dwell | 6–8 秒。隣接シーンで色・形を連続させない |
| 色 | Sassy 絵本のように原色多めの高コントラスト配色。組み合わせは複数。点滅や赤フラッシュは使わない |
| 背景クロスフェード | 約 900ms |
| 図形の動き | Canvas 2D で複数図形を毎フレーム更新。入場は 8 方向から。入場後は scene / actor ごとに決まる大振幅の 2 軸軌道でシーン領域を横断。移動に合わせた回転。シーン入場フェードと背景クロスフェードは CSS |
| 音（任意） | セッション再生中のループ BGM（もみじばミュージック）。音 ON で開始したときに 3 曲から 1 曲を選び、同一セッション中は同じ曲を継続。一時停止で pause、再開で同位置から再生、終了で破棄。タブ非表示でも pause し、再生中に再表示したら再開。シーン切替では選び直さない。初期値は OFF |
| 一時停止 | カウントダウンとシーン進行の両方を止める。タブ非表示中も Canvas ループと BGM を止める |
| 終了 | 明示操作または期限到達。リセットするまで再開しない |

## 非スコープ（MVP）

アカウント、同期、コンテンツ CMS、自動リピート、広告・課金。
音源の二次配布・販売や、音楽聴取を主目的とした配信は行わない（利用条件は [music-credits.md](../music-credits.md)）。

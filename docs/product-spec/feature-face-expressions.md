# 機能仕様: かおのゆるい変形

`colors` パックの図形に、目と口を足す。シーン dwell のあいだで
表情をゆっくり切り替える。赤ちゃんが顔に反応しやすい性質を、語りかけの対象として使う。

## 目的

- 「まる」「さんかく」などにかおを載せ、保護者が表情を言葉にしやすい状態にする
- 変形はゆるく、点滅や急な顔変わりを避ける
- どうぶつパックのイラスト顔とは独立した、図形専用の機能とする

## 位置付け

| 項目 | 内容 |
|------|------|
| 対象 pack | `colors` のみ |
| 対象状態 | `playing`（`paused` では表情タイムラインも止める） |
| 設定項目 | Setup に ON/OFF は設けない（`colors` 再生時は常に有効） |
| セッション FSM | 変更しない |
| animals との関係 | どうぶつ画像側には追加の顔レイヤを載せない |

## かおの構成

### 対象図形

| ShapeId | かお | 備考 |
|---------|------|------|
| `circle` | あり | 基準実装 |
| `square` | あり | |
| `star` | あり | 頂点が多くても目口は中央寄りに配置 |
| `triangle` | あり | 頂点を「あたま」側とし、かおは下寄りすぎない位置に |

### パーツ

| パーツ | 要件 |
|--------|------|
| 目 | 2 つ。塗りつぶしの楕円または円。白目は任意だがハイコントラストを優先 |
| 口 | 1 つ。弧または細い楕円。歯・舌は描かない |
| 色 | 図形の `foreground` に対して十分コントラストがある色（例: 図形が明るいとき濃色、暗いとき明色）。背景色そのものでの点滅はしない |
| 位置 | 図形のローカル座標に固定。転がり・回転に追従する |

鼻・耳・ほお紅・まゆげは初回スコープ外。

## 表情

初回は 2 表情のみ。

| ID | 見た目 | 語りかけの目安 |
|----|--------|----------------|
| `smile` | 目は通常、口は上向きの弧 | 「にこにこ」 |
| `surprise` | 目を少し大きく、口は小さめの開いた形（丸） | 「びっくり」 |

追加候補（非スコープ）: `sleep`（目を線に）、`neutral`。

### 切替タイミング

| 項目 | 要件 |
|------|------|
| シーン入場時 | 必ず `smile` から開始 |
| dwell 内 | シーン開始から 40–60% 経過で `surprise` へ遷移開始 |
| 遷移時間 | 700–1100ms のクロスモーフ（線形〜ease-in-out） |
| シーン終了時 | 次シーンでまた `smile` から。終了間際の強制フラッシュはしない |
| 一時停止 | 表情タイムラインも停止（現行の Canvas ループ停止に乗せる） |

隣接シーンで表情 ID を連続させない、といった制約は不要（入場が常に `smile` のため）。

## 安全制約

product-spec の安全ルールを継承し、かお固有に次を追加する。

| 禁止 | 理由 |
|------|------|
| 目の高速点滅（まばたき連打） | 点滅・てんかんリスク回避 |
| 赤目フラッシュ / 画面全体の赤点滅 | product-spec 明示禁止 |
| 表情の 200ms 未満切替 | 「チカチカ」に見えるため |
| 怖い顔・泣き顔・牙 | 短時間お遊びのトーンに合わない |

まばたきを入れる場合でも、**シーンあたり最大 1 回・片道 150ms 以上**とし、初回実装では入れないことを推奨する。

## 動きとの関係

- 既存の入場・軌道・回転は維持する
- かおは図形 path の上に同じ transform で描く
- 「ぽん」（[feature-touch-pon.md](./feature-touch-pon.md)）実装後は、スケール反応中もかおが図形に追従する

## 音

- 表情切替用の効果音はなし
- BGM 仕様は変更しない

## 実装境界（想定）

- 表情パラメータ（目の半径、口の曲率など）を `t ∈ [0,1]` で補間する純粋関数を `src/features/session/` に置く
- 描画は `draw-shape.ts`（または隣接モジュール）で path 描画の直後にかおを乗せる
- コンテンツ定数（表情定義）は `src/content/` に置いてよい
- Reducer / シーケンスの色・形選定ロジックは変えない（かおは表示レイヤ）

## 非スコープ（初回）

- 3 表情以上、またはランダム表情テーブル
- まばたきアニメ
- タッチで表情を切り替える（ぽんとは独立。将来やるなら別仕様）
- どうぶつパックへの顔オーバーレイ
- 音声合成での語りかけ

## 受け入れ条件

- `colors` 再生中、すべての図形に目と口が見える
- 各シーンで `smile` → ゆるく `surprise` へ変わり、遷移が 700ms 未満でパッと切り替わらない
- 一時停止中は表情の進行も止まる
- 点滅・赤フラッシュ・怖い顔が出ない
- `animals` 選択時（実装後）には本機能の描画が乗らない

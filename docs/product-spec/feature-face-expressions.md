# 機能仕様: かおのゆるい変形

`colors` パックの図形に目と口を足し、シーンのあいだで表情をゆっくり切り替える。
赤ちゃんが顔に反応しやすい性質を、語りかけの対象として使う。

## 目的

- 「まる」「さんかく」などにかおを載せ、保護者が表情を言葉にしやすい状態にする
- 「にこにこ」「びっくり」といった声かけのフックを、色・形に加えて用意する
- 変形はゆるく保ち、点滅や急な顔変わりで驚かせない
- どうぶつパックのイラスト顔とは独立した、図形専用の体験にする

## なぜかおか

| 選択 | なぜ |
|------|------|
| colors の図形にだけ載せる | どうぶつイラストはすでに顔がある。二重に載せて不気味・過剰刺激にしない |
| smile → surprise の 2 表情 | 「にこにこ」「びっくり」で語りかけが足りる。表情を増やしすぎない |
| シーン入場は必ず smile | 毎回やさしい顔から始まり、怖い・泣きの印象を持ち込まない |
| 遷移 700–1100ms | パッと切り替わるとチカチカに見える。ゆるい変化が声かけの余白にもなる |
| まばたきなし（初回） | 高速点滅相当のリスクと、注意が目の動きに奪われるのを避ける |
| Setup に ON/OFF なし | colors の一部として常に用意し、設定を増やさない |
| 一時停止でかおの進行も止める | 画面が止まっているのに表情だけ進むと、中断の意図と食い違う |

## 語りかけの想定

- 「にこにこしてるね」
- 「おや、びっくりしたね」
- 「あかい まるの かお」

正解やクイズにはしない。表情を一緒に見て言葉にするきっかけでよい。

## 安全との関係

| 禁止 | なぜ |
|------|------|
| 目の高速点滅（まばたき連打） | 点滅・てんかんリスク回避 |
| 赤目フラッシュ / 画面全体の赤点滅 | product-spec 明示禁止 |
| 表情の 200ms 未満切替 | 「チカチカ」に見えるため |
| 怖い顔・泣き顔・牙 | 短時間お遊びのトーンに合わない |

## 受け入れの観点

- colors 再生中、すべての図形に目と口が見える
- 各シーンで smile からゆるく surprise へ変わり、パッと切り替わらない
- 一時停止中は表情の進行も止まる
- 点滅・赤フラッシュ・怖い顔が出ない
- animals 選択時（実装後）には本機能の描画が乗らない

## 非スコープ（初回）

3 表情以上、まばたき、タッチで表情切替、どうぶつへの顔オーバーレイ、音声合成での語りかけ。

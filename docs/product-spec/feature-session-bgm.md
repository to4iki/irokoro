# 機能仕様: セッション BGM

任意のループ BGM。音楽鑑賞が目的ではなく、語りかけの場の空気を
やわらかく整えるための補助である。

## 目的

- 無音の画面だけでは場が固く感じるときに、穏やかな下地音を足せるようにする
- 音が出ることを保護者の明示的な選択に限り、不意打ちを避ける
- 音楽そのものをコンテンツ化せず、セッションの長さ・区切りを崩さない

## なぜこの扱い方か

| 選択 | なぜ |
|------|------|
| 初期値 OFF | 音は強力な刺激。必要だと判断した保護者だけが付けられるようにする |
| セッション開始時に 1 曲を選び、切替しない | シーンごとに曲が変わると注意が音に奪われ、語りかけの邪魔になる |
| 一時停止・タブ非表示で pause | 画面が止まっているのに音だけ続くと、中断の意図と食い違う |
| 終了で破棄 | 「おしまい」の区切りを音でもはっきりさせる |
| もみじばミュージック＋クレジット | 利用条件を守れる音源で、出典を Setup から辿れるようにする |

## 保護者体験としての意図

BGM は「聞かせるもの」ではなく、「そばで見守りながら話すときの背景」である。
音量や ON/OFF の最終判断は端末と保護者に委ね、アプリは勝手に大きくしない。

## 受け入れの観点

- 音 OFF のまま開始できる（初期値）
- 音 ON で開始すると BGM が流れ、一時停止で止まり、再開で続きから鳴る
- 終了後や Setup 復帰後に、意図せず音が残らない
- Setup に音源クレジットがある

## 非スコープ

曲選択 UI、シーン連動 SE、どうぶつ鳴き声、ストリーミング配信、音楽聴取専用モード。
利用条件の詳細は [music-credits.md](../music-credits.md)。

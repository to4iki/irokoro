# 機能仕様: どうぶつパック（animals）

いろパックと並ぶあそび。高コントラストな背景のうえでどうぶつのイラストを短時間見せ、
保護者が名前をゆっくり声に出すためのフックを提供する。

## 目的

- 「わんわんだよ」「ぞうさん きたね」といった語りかけのきっかけを、色・形以外でも用意する
- 赤ちゃんがシルエットで判別しやすい対象を置き、保護者の声かけと視線を重ねやすくする
- コンテンツそのものを目的化せず、あくまで人とのやり取りの補助に留める
- 既存の安全制約（短時間・点滅なし・自動リピートなし・音初期 OFF）を崩さない

## なぜどうぶつか

| 選択 | なぜ |
|------|------|
| colors と同列の選択可能な pack | 同じセッション枠のまま、語りかけのバリエーションを増やせる |
| イラスト（実写ではない） | 怖さ・過度なリアリティを避け、短時間お遊びのトーンに合わせる |
| 16 点（8 点案から拡張） | 連続セッションでも名前呼びのネタが枯れにくく、いらすとや無償枠（20 点まで）にも余裕を残す |
| 背景色は colors のパレットを流用 | 「どうぶつ」と「いろ」を同時に語れる。新しい刺激体系を増やしすぎない |
| 隣接シーンで色・どうぶつを連続させない | 単調さで注意が切れにくく、保護者も次の言葉を選びやすい |
| 回転を弱く抑える（±12° 程度） | 上下逆やスピンは認識を妨げ、語りかけの対象が「何のどうぶつか」分かりにくくなる |
| 鳴き声は初回スコープ外 | 音の不意打ちと刺激の増幅を避ける。場の音は既存 BGM に任せる |
| 初期 pack は colors のまま | いちばん説明しやすい入口を変えず、どうぶつは選択肢として足す |

## 語りかけの想定

保護者が画面を指さしながら、たとえば次のように使える状態を目指す。

- 「わんわんだね」
- 「あかい ところに ぞうさん」
- 「ペンギン みえたね」

画面は正解を求めない。名前をゆっくり言うための視覚的なきっかけでよい。

## 安全・クレジットとの関係

- 点滅や赤フラッシュは使わない。動きは colors と同系の穏やかな軌道に留める
- 音 ON でも鳴き声は出さず、既存のループ BGM のみ
- 外部 CDN に依存せず同梱する（追跡・配信依存を増やさない）
- いらすとや利用条件を守る（クレジット表記は必須ではない。詳細は [animal-credits.md](../animal-credits.md)）

## 受け入れの観点

- Setup で `どうぶつ` を選べ、開始できる
- 再生中にどうぶつが動き、隣接シーンで色・どうぶつが連続しない
- 一時停止で動きとカウントダウンが止まり、終了後は自動リピートしない
- 音の初期値は OFF。ON でも鳴き声は出ず BGM のみ

## 非スコープ（初回）

鳴き声・効果音、歩行スプライト、CMS / リモート差し替え、21 点以上の追加、
タッチ反応（[feature-touch-pon.md](./feature-touch-pon.md)）。

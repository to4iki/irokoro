# 機能仕様: どうぶつパック（animals）

保護者が赤ちゃんに語りかけながら、高コントラストな背景のうえで
どうぶつのイラストを短時間楽しむあそびパック。`colors` と同列の選択可能な pack とする。

## 目的

- Setup の「どうぶつ（もうすぐ）」枠を実装し、選択可能にする
- 色の語りかけに加え、「わんわんだよ」など名前呼びのフックを提供する
- 既存の安全制約（短時間・点滅なし・自動リピートなし・音初期 OFF）を崩さない

## 位置付け

| 項目 | 内容 |
|------|------|
| Pack ID | `animals` |
| Setup 表示 | `available: true`（現行の予約枠を解放） |
| 初期 pack | 変更しない（初期値は引き続き `colors`） |
| セッション FSM | 既存の Setup → Playing ↔ Paused → Finished を流用 |
| 時間 / 音 | `colors` と同じ（60/120/180 秒、音初期 OFF、ループ BGM） |

## コンテンツ

### どうぶつ一覧（初回）

いらすとやから DL した PNG をアプリ同梱 asset とする。商用無料枠（1 制作物あたり 20 点まで）に余裕を持たせ、**16 点**とする（初回案の 8 点から拡張）。

選定基準:

- 赤ちゃんでもシルエットが判別しやすい
- 正面〜やや斜め正面で、画面内での向きが安定している
- 武器・怖い表情・血・過度にリアルなものは避ける
- 1 動物 = 1 ファイル（ポーズ違いは持たない）

### 背景色

`colors` パックの `COLORS` を流用する。シーンは「背景色 × どうぶつ」の組み合わせ。

### シーケンス不変条件

`colors` と同様:

| 項目 | 要件 |
|------|------|
| シーン dwell | 6–8 秒 |
| 隣接シーン | 連続で同じ `colorId` にしない |
| 隣接シーン | 連続で同じ `animalId` にしない |
| 生成 | セッション開始時に deterministic なシーケンスを作成 |

## 見た目と動き

| 項目 | 要件 |
|------|------|
| 描画 | Canvas 2D。図形 path の代わりにどうぶつ画像を描画する |
| 入場 | 既存の 8 方向入場を流用 |
| 軌道 | 既存の大振幅 2 軸軌道を流用。シーン領域を横断する |
| 回転 | **抑える**（最大 ±12° 程度のゆるい傾き）。上下逆や高速スピンはしない |
| サイズ | 主アクターはシーン短辺の roughly 35–45%。副アクターはそれより小さくてもよい |
| 透過 | PNG の透明部分を活かす。白背景の矩形が見えないこと |
| フェード | シーン入場フェードと背景クロスフェードは既存 CSS を流用（約 900ms） |
| 点滅 | しない。急な白画面・赤フラッシュも使わない |

アクター数は `colors` と同様に複数可。ただし主アクター（大きいどうぶつ）を 1 体はっきり見せる。

## 音

- どうぶつ固有の効果音（鳴き声）は **初回スコープ外**
- 音 ON 時は既存のループ BGM のみ（もみじばミュージック）
- シーン切替で曲を選び直さない（現行仕様どおり）

## アセット運用（いらすとや）

### 取得と配置

1. [いらすとや](https://www.irasutoya.com/) から対象イラストを DL
2. 必要なら余白トリム・リサイズのみ行う（キャラ自体の大幅改変はしない）
3. `src/assets/animals/` に `dog.png` など ID 名で配置
4. Vite / Bun の ESM import でバンドルする（外部 CDN は使わない）

### 利用条件（仕様上の前提）

確認のたびに公式の [ご利用について](https://www.irasutoya.com/p/terms.html) を正とする。実装時点の前提:

- 著作権は放棄されていない
- 規約の範囲内で個人・法人・商用・非商用とも利用可
- 商用の無償利用は **1 制作物あたり 20 点まで**（本パックは 16 点で運用）
- 素材そのものの販売・二次配布、イメージを損なう利用は禁止
- クレジット表記は必須ではないが、音楽クレジットと同様に Setup で出典を示す

### クレジット表示

Setup 画面に次を表示する（BGM クレジットと同ブロックでも可）:

- 文言例: `どうぶつイラスト: いらすとや`
- リンク先: `https://www.irasutoya.com/`
- 定数は `src/content/animal-credits.ts`（音源 import 非依存の `music-credits.ts` と同型）

利用している素材の一覧・確認日は [animal-credits.md](../animal-credits.md) に記録する（実装時に作成）。

## UI / 文言

| 箇所 | 内容 |
|------|------|
| Setup shortLabel | `どうぶつ` |
| Setup detail | `16つのどうぶつ`（点数に追従） |
| Player title | `どうぶつを みつけよう` |
| Player description | `どうぶつのなまえを、ゆっくり声に出してみましょう。` |
| aria-label | `{pack.shortLabel}の再生画面`（既存パターン） |

一時停止・終了・残り時間表示は `colors` と同じコントロールを使う。

## 非スコープ（初回）

- 鳴き声・効果音
- どうぶつの歩行アニメ（スプライトシート）
- CMS / リモート差し替え
- 21 点以上の素材追加（無償枠を超える場合は有償手続きが必要）
- タッチ反応（別仕様: [feature-touch-pon.md](./feature-touch-pon.md)）
- 図形への顔付け（別仕様: [feature-face-expressions.md](./feature-face-expressions.md)）

## 受け入れ条件

- Setup で `どうぶつ` を選べ、開始できる
- 再生中にどうぶつ画像が動き、隣接シーンで色・どうぶつが連続しない
- 一時停止で動きとカウントダウンが止まり、終了後は自動リピートしない
- 音の初期値は OFF。ON でも鳴き声は出ず BGM のみ
- 外部 CDN や追加ネットワーク取得なしで画像が表示される
- Setup にいらすとやクレジットが出る
